# ADA3-LISTA-
¿Qué sucedió con las listas? Mando mensaje de error, Explica tus conclusiones.
Cuando eliminé los elementos repetidos, el programa marcó error porque al borrar nodos dentro de las listas enlazadas se perdieron los enlaces entre los ingredientes. Esto pasó porque al eliminar un nodo no se actualizó bien la conexión con el siguiente, y la lista se rompió. En conclusión, las listas enlazadas deben manejarse con cuidado al eliminar datos para no dañar su estructura.
